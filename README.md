1. Cloner le projet
```bash
    git clone https://github.com/FHJBC/foodzero.git
```
2. Installer les dépendances
```bash
    cd foodzero
    cd api
    yarn install
    yarn start
```
```bash
    cd foodzero
    cd client
    npm install
    npm run dev
```

```bash
    cd foodzero
    cd admin
    npm install
    npm run dev
```
